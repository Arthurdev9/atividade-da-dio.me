Como adicionar títulos e parágrafos

Utilizando as tags, você consegue adicionar o título principal da página ou subtitulo, que seria o <h2> e outros "títulos" menores que vão com uma contagem de h3 até h6. Com a tag, <p> você conseguirá adicionar parágrafos na sua página. Veja os exemplos de títulos abaixo:

Como alterar ou destacar palavras nos parágrafos

Para você alterar os destacar as formas das palavras, como adicionar um modelo itálico ou sublinhar o mesmo, são utilizadas determinadas tags, vejamos no exemplo abaixo:

  1. Este é um texto marcado.
  2. Este é um texto pequeno.
  3. Este é um texto itálico.
  4. Este é um texto sublinhado.
  5. Este é um texto em negrito.
  6. Este é um texto subscrito.
  7. Este é um texto sobrescrito
  8. Este é um texto em bloco.
  9. Este é um texto vermelho.
  10. Este é um texto riscado.
  11. Abrev

Seguindo a ordem dos exemplos mostrados acima, para fazer cada um deles é necessário usar determinadas tags, sendo elas:

  1. utilizar a tag <mark>
  2. utilizar a tag <small>
  3. utilizar a tag <i>
  4. utilizar a tag <u>
  5. utilizar a tag <strong>
  6. utilizar a tag <sub>
  7. utilizar a tag <sup>
  8. utilizar a tag <blockquote>
  9. utilizar a tag <font> Com o atributo "color" você altera a cor da palavra
  10. utilizar a tag <del>
  11. utilizar a tag <abbr>
